斐波那契螺旋

数列 1, 1, 2, 3, 5, 8, 13, 21… 每项都是前两项之和。把这些数字当边长，依次拼出正方形，在每个正方形里画四分之一圆弧——就出现了与鹦鹉螺贝壳截面几乎一样的螺旋线。

数学：递推数列 生物：贝壳生长曲线 艺术：黄金分割构图 植物：松果螺旋规律
相邻两项的比值——13/8=1.625，21/13≈1.615，55/34≈1.618——越来越接近黄金比例 φ≈1.618。φ 是方程 x²=x+1 的正数解，出现在古希腊建筑柱子比例、松果螺旋数（8条和13条，正好是相邻斐波那契数）中。不过，并非所有"发现黄金比例"的说法都严谨——其中有些只是人类的确认偏误，看到想要的就认定它在那里，你能分辨哪些是真的吗？
迭代次数 3

科赫雪花

每次把每条边的中间三分之一换成向外突出的等边三角形——重复这个操作。把滑块拉到 0 看起点；推到 5 看终点。注意边数如何爆炸式增长。

数学：无穷级数与极限 物理：海岸线悖论 自然：雪花结晶对称性 计算机：分形图形学
科赫雪花是著名"分形"：每次迭代周长乘以 4/3，趋向无穷；但面积每次增加越来越小的量，收敛到有限值。结论：无穷的边界，有限的面积。英国海岸线到底有多长？越精细的尺子量出来越长，原因与此相同——这就是曼德布罗特（分形之父）提出的"海岸线悖论"。维数介于 1 和 2 之间的"分数维"是现代混沌理论的基础。
每颗种子旋转角度 137.5°

向日葵种子排列

向日葵花盘里，每颗种子比上一颗旋转约 137.5°（黄金角）。这个角度让种子排列最紧密、不留空隙。把滑块移到 135° 或 140°，看看会出现什么破绽。

生物：植物叶序（互生叶片） 数学：无理数与最优填充 进化：自然选择的优化结果
黄金角 ≈ 137.508° = 360° × (1 − 1/φ)。关键在于：φ 是"最难被有理数近似"的无理数。因此以 137.5° 旋转的种子，任意两颗永远不会落在同一射线上，确保最大填充密度。稍微偏离（如 137° 或 138°），种子就出现明显"辐射缝"。向日葵、菊花、松果、凤梨的螺旋数几乎都是相邻斐波那契数对（8与13，13与21……）。
花瓣参数 n = 3

数学玫瑰曲线

公式只有一行：r = cos(n·θ)。其中 r 是到中心的距离，θ 是角度。n 是奇数时有 n 片花瓣，n 是偶数时有 2n 片——拖动滑块验证这个规律。

数学：极坐标方程 艺术：图案与对称设计 工程：天线辐射图案 物理：旋转波形叠加
玫瑰曲线 r = cos(nθ) 用"极坐标"描述——不是"x 向右走多远"，而是"从中心出发，转到角度 θ，走 r 远"。把 cos 换成 sin，图案旋转 90°/n。把 n 换成分数（比如 3/2），会出现需要绕两圈才能画完的更复杂玫瑰。这类曲线是工程中天线方向图、机械凸轮轮廓、芯片光刻图案的数学基础。
