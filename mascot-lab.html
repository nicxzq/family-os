角色实验室

毛毛是一只小小鸡——但她可以变成你们家的任何一个人。
为每个家庭成员上传一张照片，他们会以"小鸡毛毛 + 自己的脸"出现在故事里；
也可以让孩子画一只全新的角色，扫描后选"完整画作"——他自己的画就是主角。
照片只存在这台设备上，不上传任何地方。

怎么在新故事里用

给 Claude Code 的提示词模板（复制粘贴即可）：

请用 mascot.js 系统创作一个新的毛毛绘本：
- 主题：[XX]（比如：关于专注 / 关于读书 / 关于...）
- 主角：可以用毛毛、卡梅拉，也可以用我家庭名册里的成员（爸爸/妈妈/大宝/小宝）
- 长度：6 页，最后一页是结尾卡
- 风格：和 storybooks/01-try-it.html 完全一致
- 关键：每个角色用 <family-character who="X" pose="Y" size="N"></family-character> 调用
- 文件保存到 storybooks/0X-名字.html
- 别忘了在 for-youngest.html 里加入新绘本卡片

在 HTML 里手动写一个角色：

<!-- 必须先引入 -->
<script src="mascot.js"></script>

<!-- 然后随处使用 -->
<family-character who="毛毛" pose="reading" size="200"></family-character>
<family-character who="小宝" pose="proud" size="180" show-name></family-character>
<family-character who="卡梅拉" pose="surprised" size="160"></family-character>

属性说明：

  • who — 角色名（毛毛 / 卡梅拉 / 爸爸 / 妈妈 / 大宝 / 小宝 / 或您自定义）
  • pose — 姿势（standing / reading / shy / surprised / proud / thinking / running / sleeping）
  • size — 像素大小（默认 200）
  • show-name — 加上这个属性会在下方显示名字

原理：头像数据存在浏览器本地（localStorage），所有页面共享。同一台设备上看任何故事，都能看到您给家人设定的脸。
